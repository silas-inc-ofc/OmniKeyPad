# OmniKeypad V2.0

A simple keypad project and utilities (collection of `keypad` scripts).

## Features

- Collection of KEYPAD Python scripts
- Quick start instructions

## Designation

* E stands for Extension
* V stands for Version


## Installation

1. Clone the repo:

```bash
git clone https://github.com/silas-inc-ofc/OmniKeypad.git
cd OmniKeypad
```

2. (Optional) Create a virtual environment and install dependencies:

```bash
python -m venv venv
venv\Scripts\activate    # Windows
pip install -r requirements.txt  # if present
```

## Usage

Run one of the scripts, for example:

```bash
python "Emojikeypad E1.py"
```

Adjust the filename to the script you want to run.
